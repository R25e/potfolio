Logiciels et langages de programmation

Langages de programmation
  • Python : Utilisé pour le développement d'algorithmes et l'analyse de données complexes.
  • VBA : Langage de programmation pour automatiser les tâches dans Excel, autres applications Microsoft.
Outils d'extraction et tableaux de bord
  • QGIS : Outil de cartographie et d'analyse géospatiale pour la gestion des données géographiques.
  • Power BI : Logiciel de visualisation de données et création de rapports interactifs.
Logiciels statistiques
  • SAS : Logiciel de gestion et analyse statistique pour des traitements avancés.
  • R : Logiciel et langage de programmation dédié aux statistiques et à l'analyse de données.
  • Excel : Logiciel de tableur pour des analyses statistiques et des visualisations simples.
Systèmes de gestion de base de données
  • Oracle : Système de gestion de base de données relationnelle pour des environnements complexes.
  • Access : Système de gestion de base de données de Microsoft pour les petites à moyennes bases de données.
  • MySQL : Base de données relationnelle open-source largement utilisée dans les applications web.
  • Talend : Outil d'intégration de données permettant l'extraction, la transformation et le chargement (ETL) de données.
Outils Web
  • HTML : Langage de balisage pour structurer le contenu web.
  • CSS : Langage de style pour la présentation des pages web.
